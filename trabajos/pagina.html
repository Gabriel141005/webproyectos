Tim Berners Lee

El padre de la web

Biografia

Tim Berners-Lee nació el 8 de junio de 1955 en Londres, Inglaterra. Es un informático y el inventor de la World Wide Web, también conocida como la Web. En 1989, trabajando en el CERN, propuso la idea de una red de información que pudiera ser accesible desde cualquier parte del mundo, lo que dio origen a la web tal como la conocemos hoy.

Berners-Lee desarrolló el primer navegador web, WorldWideWeb, y el primer servidor web, httpd. Además, ideó los protocolos HTTP y HTML que permitieron la creación de sitios web. Su visión fue de una web abierta, sin restricciones, y libre de patentes.

Actualmente, Tim Berners-Lee continúa promoviendo la idea de una web accesible para todos, como director del World Wide Web Consortium (W3C), la organización que supervisa los estándares web.

Por que es importante?

Es importante porque fue la primera persona que logró establecer la comunicación entre un servidor y un cliente utilizando el protocolo HTTP, supervisó y estandarizó el desarrollo de las tecnologías necesarias sobre las cuales se encuentra fundada la Web y además que permiten el adecuado funcionamiento del internet, además de la creación del lenguaje HTML.
